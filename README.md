## 4.用户协议

1. 本项目遵循`MIT`协议，你可以自由使用，修改，分发，但是请保留原作者信息
2. 你可以选择开启`auto_report`(默认开启)
   ，轻雪会收集运行环境的设备信息，通过安全的方式传输到轻雪服务器，用于统计运行时的设备信息，帮助我们改进轻雪，收集的数据包括但不限于：CPU，内存，插件信息，异常信息，会话负载(不含隐私部分)
3. 本项目不会收集用户的任何隐私信息，但请注意甄别第三方插件的安全性
